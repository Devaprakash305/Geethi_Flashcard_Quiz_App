# 📚 Flashcard Quiz App

A simple and interactive Flashcard Quiz mobile application built using **React Native** and **Expo**. The app allows users to create, manage, and review flashcards for effective learning.

---

## 🚀 Features

- ➕ Add Flashcards
- ✏️ Edit Flashcards
- 🗑️ Delete Flashcards
- 👀 Show / Hide Answers
- ⬅️ Previous Flashcard
- ➡️ Next Flashcard
- 💾 Store Flashcards using AsyncStorage
- 📱 Simple and Responsive User Interface
- ⚡ Offline Support

---

## 🛠️ Tech Stack

- React Native
- Expo
- TypeScript
- React Navigation

## 📥 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/FlashcardQuizApp.git
```

Navigate to the project

```bash
cd FlashcardQuizApp
```

Install dependencies

```bash
npm install
```

Start the Expo development server

```bash
npx expo start
```

Scan the QR code using the **Expo Go** application.
